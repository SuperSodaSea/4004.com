# [4004.com](https://www.4004.com)

## Instruction

Install dependencies:
```bash
$ pnpm install
```

Start local dev server:
```bash
$ pnpm run dev
```

Run type check:
```bash
$ pnpm run test:types
```

Clean build cache:
```bash
$ pnpm run clean
```

Build site:
```bash
$ pnpm run build
```

Build site for distribution:
```bash
$ pnpm run dist
```

Start local preview server:
```bash
pnpm run preview
```

The build output is in the `src/.vitepress/dist` directory.
