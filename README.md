# [4004.com](https://www.4004.com)

## Usage

```bash
# Install dependencies
$ pnpm install
# Start local dev server
$ pnpm run dev
# Run type check
$ pnpm run test:types
# Clean build cache
$ pnpm run clean
# Build the site
$ pnpm run build
# Build for distribution
$ pnpm run dist
# Preview the site
$ pnpm run preview
```
